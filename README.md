## ✨ 遊戲特色

- 🏰 **9大關卡** - 對應1~9乘法表的冒險旅程
- ⚔️ **RPG升級系統** - 答題獲得經驗，等級提升至最高10級
- 🛡️ **裝備收集** - 9種獨特裝備隨等級解鎖
- 👤 **角色進化** - 裝備即時改變角色外觀
- 📱 **PWA支援** - 可安裝至手機主畫面離線遊玩
- 💾 **自動存檔** - 進度自動保存於本機

## 🎯 遊戲玩法

1. **選擇關卡** - 從1~9選擇想挑戰的乘法表
2. **回答問題** - 每關10道隨機題目
3. **升級成長** - 答對獲得經驗值，積滿升級
4. **收集裝備** - 每次升級解鎖新裝備

| 等級 | 裝備 | 類型 |
|:----:|:----:|:----:|
| 2 | 🗡️ 木劍 | 武器 |
| 3 | 🎩 皮帽 | 頭盔 |
| 4 | 👕 布衣 | 盔甲 |
| 5 | ⚔️ 鐵劍 | 武器 |
| 6 | 🛡️ 木盾 | 盾牌 |
| 7 | 🪖 鐵盔 | 頭盔 |
| 8 | 🧥 鎖甲 | 盔甲 |
| 9 | 🌟 魔法披風 | 披風 |
| 10 | 👑 王者冠冕 | 王冠 |
